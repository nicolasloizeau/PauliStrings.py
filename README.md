## Installation

From github:

```bash
pip install git+https://github.com/nicolasloizeau/paulistrings.py.git
```

##  Develop:
Install dev version
```
pip install -e .[dev]
```
Run tests
```
pytest
```
Make the documentation
```
cd docs
make html
```
