paulistrings.trim(o: Operator, max_strings: int, keep: Operator = None) → OperatorView on GitHub

Keep the max_strings Pauli terms with largest absolute coefficients, optionally keeping a specified subset of terms.

Parameters

o: Operator

The input operator to trim.

max_strings: int

Maximum number of strings to keep in the trimmed operator.

keep: Operator, optional

Operator containing terms that should be kept in the output regardless of coefficient size. Default is None.

Operator

paulistrings.add_noise(o: Operator, g: float) → OperatorView on GitHub

Add depolarizing noise that make the long string decays. g is the noise amplitude. Each string is multiplied by exp(-g * w), where w is the number of non-identity Paulis in the string.

Parameters

o: Operator

The input operator to add noise to.

g: float

The noise parameter that controls the exponential decay of coefficients.

Returns

Operator

A new operator with noise-modified coefficients.

paulistrings.k_local_part(o: Operator, k: int, atmost: bool = False) → OperatorView on GitHub

Return the k-local part of an operator (terms with exactly k or at-most k non-unit Paulis).

Parameters

o: Operator

The input operator to filter.

k: int

The Pauli lenght to filter for.

atmost: bool

If True, will keep terms with Pauli lenght <= k. If False, will only keep terms with Pauli lenght exactly = k.
